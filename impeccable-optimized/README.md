## Why v3 over v2

| Issue (v2) | Fix (v3) |
|---|---|
| Description bloat (1100 chars) | ≤400 chars structured |
| Command catalog duplicated 3× | Single source: command-metadata.json |
| Gate compression in craft (dominant failure mode admitted) | Mechanical via checkpoint.mjs |
| Asset producer Codex-only, silent degradation | Tier 1/2/3 explicit fallback |
| live.md 622L monolithic | Split playbook (≤300L) + protocol |
| Inter-command routing implicit | Uniform routing block + ROUTING.md |
| Anti-pattern adapter footgun | defineCheck() factory eliminates wiring step |

Score expected: 84% → 94%.

## Quick start

```bash
# 1. Clone Impeccable
git clone https://github.com/czlonkowski/impeccable.git

# 2. Copy v3 skill files (canonical Claude Code)
cp /path/to/impeccable-optimized/claude-code/root/SKILL.md \
   impeccable/skill/SKILL.md

cp /path/to/impeccable-optimized/claude-code/reference/*.md \
   impeccable/skill/reference/

cp /path/to/impeccable-optimized/claude-code/agents/*.md \
   impeccable/skill/agents/

# 3. Apply at least Patch 1 (gate enforcement)
cp /path/to/impeccable-optimized/patches/01-checkpoint.mjs \
   impeccable/skill/scripts/checkpoint.mjs

# 4. Rebuild
cd impeccable && bun run build

# 5. Validate
bun run test
```

See `INSTALLATION.md` for full installation paths and per-harness instructions.

## Notes for `coding-agent-skill-engineer` skill users

This overlay was produced by `/coding-agent-skill-engineer`. The pipeline:
1. Diagnostic (5-dim score)
2. SKILL.md root variants × 3 (Claude Code / Codex / Antigravity)
3. Reference variants × 3 (38 references × 3 = 114 files)
4. Systemic patches (6)
5. Routing graph + installation guide + patch notes

The overlay is **additive**: it never modifies upstream files in place. Apply selectively via the paths in `INSTALLATION.md`.
