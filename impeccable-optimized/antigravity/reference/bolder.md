# Bolder (Antigravity)

## Goal
Amplify visual impact of a bland/safe design through 5 ordered levers (color commitment, weight contrast, scale jumps, signature asymmetry, motion accent) while preserving primary task clarity and passing AI slop test.

## Autonomy boundaries
- Apply levers in order, escalating only as needed (don't apply all 5 reflexively)
- Move up color strategy axis (restrained → committed → full → drenched)
- Increase type weight contrast and scale ratio
- Introduce exactly ONE asymmetry / signature motif (not multiple)
- Run AI slop test after each major lever
- Do NOT amplify into category cliché
- Do NOT add color when committing harder to existing color works
- Do NOT apply same "bolder" pattern to brand and product registers

## Acceptance criteria
1. Color strategy advanced explicitly (named before + after)
2. Type weight contrast measurable (weight delta documented)
3. Scale ratio increased OR justified as already optimal
4. Maximum ONE new signature motif/asymmetry per pass
5. Primary user task remains obvious post-bolder
6. AI slop test 1st + 2nd order both pass after bolder
7. Register-appropriate amplification (brand = distinctiveness, product = hierarchy clarity)

## Optional checkpoints
- Confirm color strategy move if jumping more than one tier (e.g., restrained → drenched)
- Confirm signature motif choice if user has anchor references
- Otherwise: autonomous

## Inputs / outputs
- Input: target perceived as bland
- Output: diffed amplifications per lever + slop test verdict + routing

## Success metric
- 0 amplifications into category cliché
- 100% maintain primary task clarity
- 0 instances of "added 5 things" — commitment over additions

## Failure mode handling
- Lever 1 (color) sufficient → stop, skip remaining levers
- AI slop test fails post-bolder → roll back, try different lever
- Overshot (overstimulating) → STOP, suggest quieter
- Underlying composition is the issue → STOP, suggest craft re-do or critique
