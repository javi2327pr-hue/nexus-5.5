# Heuristics scoring (Antigravity special ref)

## Applicability
Consulted by: critique (always), occasionally audit for UX-adjacent reviews.

## Constraints — 5-axis rubric
1. Score each axis 1-10
2. Identify weakest axis explicitly
3. Apply register weighting (brand: emotional + craft; product: cognitive load + IA)
4. Report top 3 fixes by leverage, not severity
5. Use rubric consistently across reviews (no preference-based variance)

## Acceptance criteria
1. All 5 axis scores provided (none omitted)
2. Score derivation justified per axis
3. Weakest axis identified
4. Top 3 fixes by leverage (highest impact, not most severe)
5. Register weighting applied (brand vs product)
6. Findings actionable (specific files, specific changes)

## 5 axes
- **Visual hierarchy**: eye guided in 3s glance?
- **Information architecture**: right things right places?
- **Cognitive load**: 8 levers from cognitive-load.md
- **Emotional resonance**: POV identifiable?
- **Craft quality**: type/color/motion/spacing/states/absolute-bans?

## Score scale
- 1-3: foundation broken
- 4-6: competent but inconsistent
- 7-8: solid; minor opportunities
- 9-10: exceptional

## Failure modes
- Subjective drift across sessions → always load rubric before scoring
- All 5 axes score similarly → look closer, force differentiation
- "I don't like the colors" → not a craft score; it's preference; separate
- Brand-product preference confusion → check register before weighting
- Cannot identify weakest with confidence → re-walk the rubric per axis
