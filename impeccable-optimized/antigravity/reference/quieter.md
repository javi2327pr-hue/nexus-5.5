# Quieter (Antigravity)

## Goal
Tone down visually aggressive design through 5 ordered levers (color de-escalation, spacing increase, weight/scale compression, motion reduction, decoration removal) while preserving distinctiveness and hierarchy.

## Autonomy boundaries
- Apply levers in order, escalating only as needed
- Move down color strategy axis OR reduce chroma in place
- Increase spacing without collapsing hierarchy
- Reduce motion without killing functional motion (focus, loading, state changes)
- Do NOT strip color entirely (commitment ≠ absence)
- Do NOT apply quieter to a brand surface that should be loud (luxury isn't always quiet)
- Do NOT conflate with distill (intensity reduction vs element removal)

## Acceptance criteria
1. Color de-escalation: strategy named before + after, or chroma reduction documented
2. Spacing increase: section spacing 2x or justified as already breathable
3. Weight/scale compression: weight delta + scale ratio documented
4. Motion: decorative removed, functional preserved
5. Brand register: distinctiveness preserved (not generic SaaS)
6. Product register: earned-familiarity preserved
7. Hierarchy intact (eye still has clear path through page)

## Optional checkpoints
- Confirm color strategy move on brand surfaces (might lose identity)
- Confirm when reducing chroma below threshold (e.g., below 0.05 = essentially neutral)
- Otherwise: autonomous

## Inputs / outputs
- Input: target perceived as overstimulating
- Output: diffed reductions per lever + post-state slop test + routing

## Success metric
- 0 cases of color absence when commitment was intent
- 100% functional motion preserved
- Brand surfaces: 0 cases of lost distinctiveness
- Product surfaces: 0 cases of broken hierarchy

## Failure mode handling
- Undershot — now bland → suggest bolder
- Elements should be removed not just toned → suggest distill (different operation)
- Type now needs to carry more hierarchy → suggest typeset
- Cannot reduce intensity without losing identity → STOP, explain, ask user
