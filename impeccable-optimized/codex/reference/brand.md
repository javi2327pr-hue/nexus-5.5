# Brand register (Codex)

```
WHEN LOADED: SKILL.md Setup, when register = brand
APPLIES TO: brand sites, landing pages, marketing, campaigns, portfolios, long-form, about pages

STANCE: communicate, not transact
GENRE SPAN: tech / luxury / consumer / creative — share stance, diverge in aesthetic lane

BRAND SLOP TEST:
  If "AI made that" without doubt → failed
  Bar = distinctiveness ("how was this made?" not "which AI?")
  Restraint without intent = mediocre

SECOND SLOP TEST — aesthetic lane:
  Name the reference. Lanes:
    Klim-specimen / Stripe-minimal / Liquid-Death-acid-maximalism
    Editorial-typographic / Brutalist-utility / Industrial-signage
    Mid-century-postwar / Bauhaus-grid / Y2K-nostalgic / Vintage-product
  Don't drift into editorial-magazine on non-editorial brief.

REFLEX-REJECT AESTHETIC LANES (currently saturated):
  - Editorial-typographic (display serif italic + mono labels + ruled separators)
  - AI-dark-mode-with-purple
  - Healthcare-cream-and-teal
  - Crypto-neon-on-black

EXCEPTION: existing brand identity already committed → preserve, don't second-guess.

TYPOGRAPHY:
  - Procedure per reference/typography.md
  - STRICT reflex-reject list (Fraunces, Cormorant, Inter, Plex, etc.)
  - Pairing by brand type:
      Editorial/luxury:  display serif + sans body
      Tech/dev/fintech:  one committed sans
      Consumer:          humanist sans + script/display serif
      Industrial:        geometric sans + mono for technical
      Cultural/arts:     one type-as-identity

COLOR:
  Per reference/color-and-contrast.md
  Can go Committed, Full palette, Drenched freely
  Restrained = opt-in for brand minimalism, not default

LAYOUT:
  Editorial composition
  Asymmetry allowed
  Vary rhythm aggressively
  Custom > grid-conformance

MOTION:
  Signature moments OK
  Scroll-driven reveals
  Hero entrance

IMAGERY:
  Image-led briefs (restaurants, hotels, magazines, food, travel, fashion, product)
  → REAL or sourced imagery
  → DON'T replace with CSS scenery / fake metrics / decorative panels / filler

BRAND-SPECIFIC FAILURES:
  ❌ Category-default palette
  ❌ Reflex-reject aesthetic lane
  ❌ Editorial-magazine treatment on non-editorial brief
  ❌ Pure restraint without POV (mediocre)
  ❌ Generic "modern" type pairing
  ❌ AI-marketing-template aesthetic
```
