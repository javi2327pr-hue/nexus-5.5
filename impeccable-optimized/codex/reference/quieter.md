# Quieter (Codex)

```
5 levers in order:

Step 1: Color de-escalation
  Drenched → Full → Committed → Restrained
  OR keep strategy but reduce chroma (oklch 0.2 → 0.12 → 0.08)

Step 2: Spacing increase
  Section spacing scale = 2x current vertical rhythm
  Let things breathe

Step 3: Weight + scale compression
  Reduce weight contrast between hierarchy steps
  Compress scale ratio

Step 4: Motion reduction
  Remove decorative animations
  Keep functional (loading, focus, state changes)
  Reduce stagger if 5+ items
  Shorter durations on remaining motion

Step 5: Removal (different intent — overlaps with distill)
  If chrome to cut → suggest /impeccable distill instead
  Quieter reduces INTENSITY; distill REMOVES elements

Step 6: Register-aware
  Brand:   editorial restraint, type carries identity
  Product: earned-familiarity, predictable

Step 7: "Went too quiet" check
  Lost distinctiveness (brand)?
  Hierarchy collapsed?
  If either → /impeccable bolder

Step 8: Route
  undershot — now bland       → /impeccable bolder
  elements should be removed  → /impeccable distill
  type now carries hierarchy  → /impeccable typeset
  final pass                  → /impeccable polish
```

## Anti-patterns
```
❌ Quieter = strip color entirely (want commitment, not absence)
❌ Quieter on brand that should be loud
❌ Conflate with distill (different operation)
❌ Remove all motion (focus rings, loading = bad UX)
```
