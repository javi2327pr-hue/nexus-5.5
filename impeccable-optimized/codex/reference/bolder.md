# Bolder (Codex)

```
5 levers in order of impact:

Step 1: Color commitment (move up the strategy axis)
  Restrained → Committed (one color carries 30-60%)
  Committed → Full palette (3-4 named roles)
  Full palette → Drenched (surface IS color)
  Reflex: usually COMMIT harder to one color, don't add more

Step 2: Typography weight contrast
  Increase ratio between display + body weights
  300/900 reads bolder than 400/700
  Add italic for additional contrast plane
  Brand: introduce display serif (or vice versa) — avoid reflex-reject list

Step 3: Scale jumps
  Increase modular scale ratio: 1.25 → 1.333 → 1.5
  Bigger headlines, smaller secondary, compress the middle

Step 4: Asymmetry + signature motif
  Introduce ONE deliberate asymmetry
  Off-center hero, oversized first letter, full-bleed photo escaping grid
  STOP AT ONE

Step 5: Motion accent
  If static: add ONE motion moment
  Entrance stagger, scroll reveal, hover with personality
  Don't blanket-animate

Step 6: "Went too far" check
  AI slop test 1st + 2nd order pass?
  Primary user task still obvious?
  If either fails → /impeccable quieter

Step 7: Register-aware bolder
  Brand:   go for distinctiveness, aesthetic-lane commitment
  Product: clearer hierarchy, not visual noise

Step 8: Route
  overshot                    → /impeccable quieter
  composition is the problem  → /impeccable craft or critique
  final pass                  → /impeccable polish
  motion specifically         → /impeccable animate
  color specifically          → /impeccable colorize
```

## Anti-patterns
```
❌ Add color when committing harder would work
❌ Amplify into category cliché
❌ 5 signature motifs (pick one)
❌ "Bolder" on product = visual noise
```
