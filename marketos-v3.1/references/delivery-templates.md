# Delivery Templates — Formatos de Entrega

Templates para los bloques de insight y el reporte final de MarketOS.

---

## Bloques de insight (usar en TODO el análisis)

```
[PATRÓN DETECTADO 🧠]
  Descripción: [qué descubrimos]
  Implicación: [qué significa para la estrategia]
  Acción: [qué hacer con este insight]

[QUICK WIN ⚡]
  Qué: [acción concreta]
  Cómo: [pasos en 1-2 líneas]
  Cuándo: [ejecutable en <7 días]
  Impacto esperado: [métrica afectada]

[ALERTA ⚠️]
  Riesgo: [qué podría salir mal]
  Probabilidad: [Alta/Media/Baja]
  Mitigación: [cómo prevenirlo]

[DATO CLAVE 📊]
  Estadística: [número + contexto]
  Fuente: [de dónde viene]
  Relevancia: [por qué importa para este cliente]
```

---

## Template del resumen ejecutivo (siempre al final)

```
📋 RESUMEN EJECUTIVO
(máximo 6 líneas, sin bullets, solo prosa directa)

Línea 1: Situación actual del cliente en el mercado
Línea 2: Hallazgo principal del análisis competitivo
Línea 3: Patrón de compra más accionable detectado
Línea 4: Posicionamiento recomendado en una frase
Línea 5: Acción más importante de los primeros 30 días
Línea 6: Métrica objetivo para los 90 días
```

---

## Template de próximos pasos (siempre al final)

```
📌 PRÓXIMOS PASOS
1. [Acción inmediata — hoy mismo]
2. [Acción semana 1 — setup inicial]
3. [Acción mes 1 — primer milestone]
4. [Acción mes 3 — evaluación de resultados]
```
